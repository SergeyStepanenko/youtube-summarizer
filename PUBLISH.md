# 🚀 Быстрая публикация на GitHub

## Шаги для публикации:

### 1. Создайте репозиторий на GitHub
- Перейдите на https://github.com/new
- Название: `youtube-summarizer`
- Описание: `Python script to extract and summarize YouTube video transcripts`
- Выберите **Public**
- ❌ НЕ добавляйте README, .gitignore, LICENSE (они уже есть)
- Нажмите "Create repository"

### 2. Выполните команды в терминале:

```bash
# Добавьте все файлы
git add .

# Создайте первый коммит
git commit -m "Initial commit: YouTube transcript extractor with summarization"

# Добавьте удаленный репозиторий (замените YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/youtube-summarizer.git

# Отправьте код
git push -u origin main
```

### 3. Обновите README.md
После публикации замените в README.md:
- `yourusername` → ваш GitHub username
- `https://github.com/yourusername/youtube-summarizer.git` → реальный URL

### 4. Добавьте Topics на GitHub
На странице репозитория нажмите на шестеренку рядом с "About" и добавьте:
- `youtube`
- `transcript`
- `summarization`
- `python`
- `nlp`
- `russian`

## ✅ Готово!

Ваш проект опубликован! 🎉
