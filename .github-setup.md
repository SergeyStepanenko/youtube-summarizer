# Инструкция по публикации на GitHub

## Шаг 1: Создание репозитория на GitHub

1. Перейдите на [GitHub](https://github.com)
2. Нажмите "New repository" (зеленая кнопка)
3. Заполните данные:
   - **Repository name**: `youtube-summarizer`
   - **Description**: `Python script to extract and summarize YouTube video transcripts with Russian language priority`
   - **Public/Private**: выберите Public
   - ❌ **НЕ** создавайте README, .gitignore или LICENSE (они уже есть в проекте)
4. Нажмите "Create repository"

## Шаг 2: Подготовка локального репозитория

Выполните следующие команды в терминале:

```bash
# Добавьте все изменения
git add .

# Создайте первый коммит
git commit -m "Initial commit: YouTube transcript extractor with summarization"

# Переименуйте ветку в main (если нужно)
git branch -M main

# Добавьте удаленный репозиторий (замените YOUR_USERNAME на ваш GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/youtube-summarizer.git

# Отправьте код на GitHub
git push -u origin main
```

## Шаг 3: Настройка репозитория на GitHub

После публикации:

1. **Добавьте Topics** (теги):
   - Перейдите на страницу репозитория
   - Нажмите на шестеренку рядом с "About"
   - Добавьте topics: `youtube`, `transcript`, `summarization`, `python`, `nlp`, `russian`

2. **Обновите README**:
   - Замените `yourusername` на ваш GitHub username в ссылках
   - Можете добавить скриншоты или GIF с примером работы

3. **Настройте GitHub Pages** (опционально):
   - Settings → Pages
   - Можно создать красивую landing page для проекта

## Шаг 4: Дополнительные улучшения (опционально)

### Добавить GitHub Actions для тестов

Создайте файл `.github/workflows/test.yml`:

```yaml
name: Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: [3.8, 3.9, '3.10', '3.11', '3.12']
    
    steps:
    - uses: actions/checkout@v3
    - name: Set up Python ${{ matrix.python-version }}
      uses: actions/setup-python@v4
      with:
        python-version: ${{ matrix.python-version }}
    - name: Install dependencies
      run: |
        python -m pip install --upgrade pip
        pip install -r requirements.txt
    - name: Test import
      run: python -c "from youtube_transcript import get_transcript, get_video_id"
```

```markdown
## Как внести вклад

1. Форкните репозиторий
2. Создайте ветку (`git checkout -b feature/amazing-feature`)
3. Внесите изменения
4. Закоммитьте (`git commit -m 'Add amazing feature'`)
5. Запушьте (`git push origin feature/amazing-feature`)
6. Откройте Pull Request

## Стиль кода

- Следуйте PEP 8
- Добавляйте docstrings к функциям
- Комментируйте сложную логику

## Тестирование

Перед отправкой PR убедитесь, что:
- Код работает с Python 3.8+
- Нет синтаксических ошибок
- Функции работают корректно
```

## Полезные команды Git

```bash
# Проверить статус
git status

# Посмотреть изменения
git diff

# Отменить изменения в файле
git restore <file>

# Посмотреть историю коммитов
git log --oneline

# Создать новую ветку
git checkout -b feature-name

# Переключиться на другую ветку
git checkout main

# Обновить локальную копию
git pull origin main
```

Если хотите использовать анонимный email:
```bash
git config user.email "username@users.noreply.github.com"
```

Замените `username` на ваш GitHub username.
